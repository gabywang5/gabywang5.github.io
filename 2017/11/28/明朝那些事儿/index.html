Kindle 读书 List --《明朝那些事儿》

发表于 2017-11-28 | 分类于 书籍 | | 阅读次数:

《明朝那些事儿》总共 7 部！终于看完了！

为什么会读这套书

《明朝那些事儿》其实很早就想看了！作为一个理工科的人，看到历史真的就头疼！这不怪历史，要怪就怪很多中国老师太喜欢把历史讲的让我 kill myself 了。想当年我初中高中，数理化能考满分，但是历史却永远拖后腿。某种程度上，我觉得自己是因为极度讨厌文科而转向读理科。然而一直觉得自己人文知识匮乏，亟需补一下历史政治方面的知识，又知道「当年明月」写的书真的出名好久了，写的历史浅显易懂，适合我这种不喜欢看晦涩难懂历史的人～

今年夏天6月份买了 Kindle，548 RMB，想着以前一直想看但却没看的《明朝那些事儿》该看得下去了，毕竟光是买 Kindle 就花了这么些钱，要多看完点书来回本才是！（相信这也是很多人买 Kindle 的初衷吧。。想通过买 Kindle 来激励自己看书，毕竟不多看点书又是一波血亏啊。。）

看完书的感想

《明朝》总共有 7 部，从朱元璋由一个乞丐被元朝逼地走投无路奋起反抗、打败了无数枭雄、经历了几十年的腥风血雨、最终建立了寿命长达 270 年的明朝；到最后王朝陨落，看完以后心里着实像过了几个世纪。

感想有很多，更多的是看书期间，看到一些关键的转变的时候，对于人的信念和做事方法的收获，收获的是一些大智慧。

感想待更新。

Kindle 读完的书的 List：

  1. 《明朝那些事儿》.当年明月
  2. 《认知突围：做复杂时代的明白人》.蔡垒磊
  3. 《Letter from an Unknown Woman》.Stephan Zweig
  4. 《逻辑思维》.罗振宇
