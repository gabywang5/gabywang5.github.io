认识 RS 触发器。
第一步：
It “remembers“ which of two inputs was last a voltage. 就是能“记住”上一个为 1 的电压是哪一个。

第二步：
It is a circuit that remembers whether a particular signal was 0 or 1 at a particular point in time.
