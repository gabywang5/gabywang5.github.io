虽然是照着教程敲的，但是调试这么个小程序都还有几个要注意的地方。
由时间顺序来讲。

首先，代码语法错误。第一次调试出现错误提示 importError，找不到 MINEText module，我找啊找，查啊查，最后坑爹的发现自己把 MIMEText 打成了 MINEText，真的是一口老血吐出来…

然后，确定没有语法错误以后，提示图中错误：

看提示信息，说是 smtp 验证错误，需要一个安全链接，比如 ssl，于是把原教程中的 server = smtplib.SMTP(smtp_server, 25)，改成了 server = smtplib.SMTP_SSL(smtp_server, 25)，心里想着，好，你让我用安全链接去验证，那我用，这下没问题了吧…

谁知，又提示：

纳尼？…这又是什么幺蛾子？于是查啊查，发现是 smtp 的 ssl 加密方式有特定的服务器端口号。

其实，在配置 QQ 邮箱的 ssl 的时候，根据 QQ 邮箱提供的方式(如何设置 POP3/SMTP 的 SSL 加密方式？http://service.mail.qq.com/cgi-bin/help?subtype=1&&id=28&&no=369)，已经可以得知，端口需要用 465 或者 587,这两个我都试了一下，465 ok，587 fail。

接着又会需要授权码。（什么是授权码，它又是如何设置？http://service.mail.qq.com/cgi-bin/help?subtype=1&&id=28&&no=1001256\r\n%27）
个人理解，授权码是对通过第三方客户端登陆本方邮箱的一种安全验证。至于为什么非要一个授权码，留坑后面再填…

至此，发一个很 挫 简单的邮件成就达成，可以继续下一步的精致工作了～
