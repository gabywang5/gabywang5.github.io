python学习归纳总结

发表于 2018-06-04 | 分类于 python | | 阅读次数:

python 语法知识学习笔记。

作用域

函数和变量的作用域分两种：
1、 可以直接引用，包括 public的 和 一部分特殊变量。public 的栗子，比如：abc,PI等等；一部分特殊变量包括，模块里面的 __author__,__name__
2、 不应该被直接引用的，private。类似_xxx,或者__xxx

private函数不被直接引用的话，可以用来隐藏函数细节，属于代码封装跟抽象的方法之一。见 https://www.liaoxuefeng.com/wiki/001374738125095c955c1e6d8bb493182103fac9270762a000/0013868200196665403ac40fac14536939dd5af20810782000

安装第三方模块

现在试着安装第三方库 python image library(PIL),PIL 已经停止更新，取而代之的是 Pillow。
pip install Pillow
如果出现权限错误提示，那么 sudo 即可
sudo pip install Pillow
有必要的话，可以更新 pip 的版本
pip install –upgrade pip

兼容新版本

python 的更新比较频繁，如何在旧版本中能够使用新版本的特性？可以使用 __future__
例如：
from __future__ import unicode_literals
from __future__ import division

变量可访问性

xxx 可以从外部直接访问
_xxx 可以从外部直接访问，但是请尊重一下我，没事不要随意访问
__xxx 不好意思，不可以从外部直接访问
__xxx__ 系统变量 可以从外部直接访问
第三种情况，双下划线开头的变量是否一定不可以从外部直接访问呢？其实也不是，因为不同的python 解释器会把__xxx解释为不同的变量名。
