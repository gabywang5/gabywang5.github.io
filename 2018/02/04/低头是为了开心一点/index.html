低头是为了开心一点

发表于 2018-02-04 | 分类于 进步 | | 阅读次数:

此篇为了记录一件小事。最然只是一件小事，但是于我而言，是非常有意义的值得记录的事情。

某种程度上，我是一个没有什么耐心的人。与人交谈，如果并没有相谈甚欢，甚至是发生了一些争执，那么我可能很容易就放弃了继续谈话下去的可能，因为理所当然地无理由地坚持自己的观点。我认真地剖析过了自己，觉得自己的性格有很大改善的空间。现在的自己有一点点的以自我为中心，宁愿与别人「撕破脸」，也不愿「低头」认错，让步。其实很多时候，发生争执的起因可能只是一件微不足道的事情，比如语气不够礼貌、表现得不够耐烦等等。一方低头认错，另一方可能就释怀了，大家喜笑颜开，开开心心，何乐而不为。没必要为了一些没意义的小事闹得自己不愉快、影响自己的心情。毕竟啊，开心最重要！

尤其是，当发生争执的对象是你在未来或者生活中不得不面对的、逃避不了的人，对方又是死不认错、不会主动低头的主，那此时更有必要主动低头了！

民间有句谚语，‘低头的是稻穗，抬头的是稗子’。虽然用在此处不是特别恰当，但是大致都在告诉我们，要做一个内心谦逊、充实，懂得低头的人。低头，是为了不让沙子进了眼睛。

希望自己一步步地完善性格。
