第一个 Web 应用程序总结

发表于 2018-07-19 | 分类于 python | | 阅读次数:

第一个 web 应用程序，选择了自己很感兴趣的网站设计，项目主要是一个学习笔记记录网站，功能非常精剪，包括：1、用户的注册、登录、登出；2、主题、条目的增删改。在本地确定能 work 之后，将其部署到服务器，以使任何有网络的用户都能访问和使用网站。

项目在本地能够 work 了，但是后面将其部署到 Heroku 的时候，出现了问题，折腾了半天估计是 runtime.txt 的 python 版本要求问题，中间重装了python版本以后，再次运行部署的命令，又出现了显示其他 ModuleNotFoundError 的错误，最终第一个项目还是未能成功部署。

虽然出师不利，但是在拿本项目练手的过程中，有几点心得：
1、开发一定要从简单到负责，每一个新功能、新框架的应用，都是要在确保前面所做工作没有问题的基础上才能进行，这样才能扎扎实实的推进，而不是最后 fail了，但是却不知道中间到底哪一个环节出了问题。
2、要想开发出既实用又美观的 web 应用程序，需要多方面的知识，包括 python、javascript、web框架、css、服务器等等…
3、多敲代码才能真正理解 how they exactly work。

总而言之，继续学习！
