抓娃娃机抓娃娃心得体会

发表于 2018-07-03 | 分类于 life | | 阅读次数:

这题目取的也是有点密集恐惧症…最近沉迷于抓娃娃无法自拔，从一开始的小白到现在 get 到了一点经验，看到抓到的娃娃这么多这么卡哇伊，不能就让它们晾在那里啊，一定要 得瑟得瑟 晒出来，跟大家 炫耀 分享一下我的心得体会。

首先，这段时间的抓娃娃的地点简直颠覆我的三观，当然地点也是在商场…但是我从来没有见过有哪个商场的娃娃机这么多人！👇 要不要这么火爆啊！这还是下午四五点的时候，不是吃饭时间，就算不是周末，人也不少！要知道平时我见到的娃娃机真的是门可罗雀啊，每次路过娃娃机最多只看到零星几个人在那儿蹭蹭就走了。这个商场的娃娃机基本不缺人，而且超级有欢乐的氛围！

炫耀 show一下我的战果。实际的娃娃看起来要比图片上大得多。需要说明的是，这些娃娃都是以市场售价的1/3的成本抓到的，所以还是非常有成就感哒～如果你抓娃娃的技术不行，那么很有可能和我看到的许多败北的小伙伴一样，跟一个娃娃在那死磕，最后用了二三十个币，只夹到了一个售价七八块的娃娃。这个时候，我想，得到的不是欢乐，而是委屈吧。。。不过，不排除有些人喜欢那种即使不惜一切代价也要死磕到底最终成功的畅快感！

好了，说正题，怎么 很厉害地 快速有效地抓到娃娃？总结了以下几点：
1、选定合适的娃娃机
决定你是否抓到娃娃的根本不是你有多厉害，而是一个娃娃机的抓力是否够大。你再怎么是娃娃杀手，碰到一个疲软的娃娃机那也是要凉凉的🤷‍♀️。去到一个娃娃机，先别急着抓，先观察观察正在抓的人，娃娃机的爪子抓到公仔的时候，是否能够基本保持一定的抓力移动，而不是刚碰到，下一秒就松开了，这样的娃娃机就是来坑钱让你心塞的，不会给你带来任何乐趣和成就感。为什么说是保持一定的抓力移动而不是保持 100% 抓力移动呢？因为有些公仔的表面材料很光滑，有些表面毛茸茸的，毛绒的又有光滑的毛和炸毛，炸毛基本是最好抓的那一类，因为摩擦力够大，爪子不容易“手抖”让其溜走。
这类兔子很好抓，因为表面比较粗糙，不是那种滑溜溜的毛

这种就比较难抓了，滑溜溜的，很容易从爪子上掉下来

2、判断娃娃的体型
最近粉红豹好像挺流行，但是其实粉红豹是最难抓的那一类。小猪佩奇、米奇、哆啦A梦这些更容易抓到。因为当机器的爪子抓到娃娃的时候，会紧接着垂直向上移动，再沿水平方向向洞口移动。想象爪子抓住的是一个球，在这个移动的过程中，爪子要想一直抓住球不掉，那当它往自己球心位置发力抓紧的时候，球可不能瘪了吧，要是球瘪了，或者表面缺一块，那很容易爪子其中的一两根就会抓空了，然后球就掉了。我暂时还没想到更好的比喻，大致要说明的意思就是，如果爪子抓住的部分，不是很胖、很丰满、（好像有什么不对…）很实在的部分，那在它垂直向上移动并同时发力继续抓紧的时候，就会容易扑了个空，抖一下，就掉了。具体说明就是，粉红豹四肢和躯干都比较细长瘦，那么虽然一开始抓住了，但是后来爪子垂直向上，继续抓紧，四肢和躯干被压缩，体积变小，爪子就容易没抓稳。但是像小猪佩奇，如果你一开始对准它的屁股下半部分也就是屁股部分抓紧了，那当它继续发力，爪子向内 push 的时候，娃娃是不会被挤瘪的，因为屁股部分都填充的很满，没有多余的空隙被压缩。这样，娃娃在移动的过程中，就不容易中途掉下来。类似的，对准米奇、哆啦A梦的大头抓，可以减小中途掉下的概率。
哆啦A梦和大雄这类体型比较好抓。

粉红豹不好抓（实在抱歉去拍照的时候，之前娃娃机里面的小 size 的粉红豹已经没了，这是放在机器外面展览的巨型粉红豹，可以想象一下它变小以后，应该是比较纤细的，不好抓）

3、地势很重要
此地势非彼地势。这里，地势是指，从水平面看过去，娃娃所处的高度在垂直方向上，相对洞口高出的那段距离。比如，下面图一的娃娃处于地势高的地方，图二的娃娃处于地势低的地方。地势高的娃娃，会比地势低的娃娃更容易夹到！地势低的娃娃，即使一开始夹住了，在向洞口平移的过程中，要么会碰到比它地势高的娃娃的手脚和躯干，陨落…要么在快要到达洞口的时候，差一点点翻出洞口，掉出来，就又顺着地势低的方向滚回原来被夹起的地方，于是你就开始了一个夹过来、滚回去、夹过来、滚回去的一个死循环…

4、选定合适的时机
去抓娃娃最好选择人多热闹的时候，因为要想持续快速的抓到娃娃，那必须有足够的人来创造可以抓的娃娃，也就是具备上面 2、3 两种条件的娃娃。但是一个娃娃机往往好抓的就一个，那么这最好抓的一个出洞口以后，怎么才能继续抓呢？答案就是，先走开，让别的小伙伴去试试水，去折腾一下，过一会儿就又会出现潜力股可以让你抓啦。😏

5、不要吊死在一个娃娃上
有的时候，你很喜欢的一个娃娃就是抓不出来怎么办呢？别钻牛角尖，想着老子今天不把你抓出来誓不为人，这样大概率你最后还是抓出来了，毕竟人民币玩家还是有的😂，但是可能花了几十个币，也还是不会太开心吧…最好就是，如果想抓一个娃娃，先试两三次，如果两三次以后不行，及时 pauss，走开一会儿，去看看别的娃娃有没有在等着你，说不定抓到了其他的娃娃以后回来看，咦？刚才的娃娃被别的小伙伴折腾过后更好抓了呢！

6、保持平常心
抓不到娃娃的时候不要灰心丧气，觉得自己的钱石沉大海了，就算抓不到，那你也获得了玩耍的乐趣，并不是一无所获。既然出来抓娃娃了，那就是花钱来玩来买开心了，这跟你花钱去 ktv 唱歌是一样的。讲真，娃娃的成本并不高，你完全可以花钱直接去买到一模一样的娃娃。但是，试想一下，直接买来的娃娃，开心吗？娃娃机的意义就在于，通过自己的智慧、动手能力，得到了这个娃娃，这个过程体现的是你对对手、对形势绝佳的判断能力（我吹不下去了…），可以带来成就感和动手的乐趣。

祝看完这篇 闲文 心得的小伙伴们以后抓娃娃都一抓一个准🐯。
